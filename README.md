# OWASP-Top-10-2025---Examples-of-vulnerabilities
Здесь представлены примеры уязвимостей некоторых категорий OWASP Top 10:2025, над которыми была проведена небольшая аналитическая работа (способы обнаружения, методы устранения и т.д.).

Цель данного репозитория — систематизировать информацию о реальных уязвимостях, описанных в базе CVE, и сопоставить их с категориями OWASP Top 10. Для каждой уязвимости был проведён краткий анализ, включающий описание проблемы, причину её возникновения, возможные последствия, а также общие методы обнаружения и устранения.

В каждом файле рассматривается отдельная уязвимость. Анализ строится по следующей структуре:

- общая информация об уязвимости (CVE, продукт, категория OWASP, тип CWE);
- краткое описание уязвимости;
- причина возникновения;
- возможные последствия;
- концептуальный сценарий эксплуатации;
- способы обнаружения;
- меры по устранению и снижению риска;
- источники информации.

Основными источниками информации при подготовке анализа являются официальные базы уязвимостей и стандарты безопасности, такие как:
- National Vulnerability Database (NVD)
- MITRE CWE
- OWASP Top 10

Данный репозиторий носит учебно-аналитический характер и предназначен для демонстрации понимания базовых принципов анализа уязвимостей и их классификации в рамках OWASP Top 10.
